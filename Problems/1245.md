Question 1245
==================
문제 설명
------------------
다음과 같은 n*m 배열 구조를 출력해보자.

입력이 3 4인 경우 다음과 같이 출력한다.
1 3 6 9
2 5 8 11
4 7 10 12

입력이 4 5인 경우는 다음과 같이 출력한다.
1 3 6 10 14
2 5 9 13 17
4 8 12 16 19
7 11 15 18 20

입력이 n m인 경우의 2차원 배열을 출력해보자.

입력
------------------
첫 번째 줄에 배열의 크기 n m이 입력된다.
[입력값의 정의역]
1<=  n,m <= 100

출력
------------------
n*m 크기의 배열을 설명과 같이 채워 출력한다.

입력예시
------------------
2 3

출력예시
------------------
1 3 5 
2 4 6 

Answer
==================
```cpp
#include <stdio.h>

int main()
{
    int n, m, a[101][101], k=1;

    scanf("%d %d", &n, &m);

    for(int p=1;p<=n+m;p++)
    {
        for(int j=1;j<=m;j++)
        {
            for(int i=1;i<=n;i++)
            {
                if(i+j==p+1){ a[i][j]=k; k++;}
            }
        }
    }

    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            printf("%d ", a[i][j]);
        }
        printf("\n");
    }
}
```
