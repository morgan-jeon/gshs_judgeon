Question 1243
==================
문제 설명
------------------
다음과 같은 n*m 배열 구조를 출력해보자.

입력이 3 4인 경우 다음과 같이 출력한다.
10 9 4 3
11 8 5 2
12 7 6 1

입력이 4 5인 경우는 다음과 같이 출력한다.
20 13 12 5 4
19 14 11 6 3
18 15 10 7 2
17 16 9 8 1

입력이 n m인 경우의 2차원 배열을 출력해보자.

입력
------------------
첫 번째 줄에 배열의 크기 n m이 공백을 두고 입력된다.
[입력값의 정의역]
1<= n,m <= 100

출력
------------------
n*m 크기의 배열을 설명과 같이 채워 출력한다.

입력예시
------------------
입력1
1 3

입력2
2 3

출력예시
------------------
출력1
3 2 1

출력2
6 3 2
5 4 1 

Answer
==================
```cpp
#include <stdio.h>

int main()
{
    int a[100][100];
    int k=1;
    int n, m;

    scanf("%d %d", &m, &n);

    for(int i=1;i<=n;i++)
    {
        if(i%2==1){
        for(int j=1;j<=m;j++)
        {
            a[i-1][j-1]=k;
            k++;
        }
        }

        else{
        for(int j=1;j<=m;j++)
        {
            a[i-1][m-j]=k;
            k++;
        }
        }
    }

    for(int i=m;i>=1;i--)
    {
        for(int j=n;j>=1;j--)
        {
            printf("%d ", a[j-1][i-1]);
        }
        printf("\n");
    }
}
```
