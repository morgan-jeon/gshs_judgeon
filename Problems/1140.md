Question 1140
==================
문제 설명
------------------
입력되는 양의 정수를 뒤집어 출력하라

Example 1

input

120

output

21

입력
------------------
n이 입력으로 들어온다(0<=n<=231-1인 정수)

출력
------------------
n의 값을 뒤집어 출력한다.

입력예시
------------------
123

출력예시
------------------
321

Answer
==================
```cpp
#include <stdio.h>

int main()
{
    int a, k=0, o;
    scanf("%d", &a);
    while(a%10==0)
    {
        a/=10;
    }
    while(a/10!=0)
    {
     printf("%d", a%10);
     a/=10;
    }
    printf("%d", a);
}
```
