Question 1135
==================
문제 설명
------------------
양의 정수 n이 입력된다.

이때 n을 이진수로 변경하여 1의 개수를 출력하고자 한다.

입력
------------------
n이 입력된다(1<=n<=10,000,000)

출력
------------------
n을 이진수로 변경할 경우 1의 개수를 출력하라

입력예시
------------------
5

출력예시
------------------
2

Answer
==================
```cpp
#include <stdio.h>

int main(){
    int n, a[30]={}, i, k=1, r;
    scanf("%d", &n);
    for(i=2;i<=n;i*=2)
    {
        k+=1;
    }
    for(i=1;i<=k;i++)
    {
        a[i]=n%2;
        n/=2;
    }
    for(i=k;i>=1;i--)
    {
        if(a[i]==1)
        {
            r+=1;
        }
    }
    printf("%d", r);
}
```
