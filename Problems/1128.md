Question 1128
==================
문제 설명
-----------------------------
@(골뱅이)를 좋아하는 gt는 매일매일 골뱅이를 먹는다.

지금까지 gt가 먹은 골뱅이를 출력하라

입력
-----------------------------
입력으로 지금까지 먹은 골뱅이의 개수 N이 입력된다.

1 <= N <= 210 인 정수

출력
-----------------------------
N개의 @를 출력한다.

입력예시
-----------------------------
5

출력예시
-----------------------------
@@@@@


Answer
==================
```cpp
#include <stdio.h>
int main()
{
    int n, m=0;
    scanf("%d", &n);
    while(m<n){
    printf("@");
    m++;
    }
}
```
